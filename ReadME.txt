Project Name: BankingSystem

Project funtionalities:
* Allows users to sign in
* Allows new users to sign up
* Allows customers who have signed in to make transactions (lodgement, withdrawal, transfers)
* Allows customers who has signed in to view balances

Database:
* Created in mySQL
* Hosted in gearhost

Project Components:
* Server - BankingSystem
* Client - BankingClient (includes all the GUIs)

Usage:
1. Build BankingSystem
2. Run BankingSystem
3. Build BankingClient
4. Run BankingClient
5. The GUIs should pop up
6. To exit, click on the "x" button on the top right corner
